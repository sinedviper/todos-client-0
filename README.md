# Todos-Client

### :pushpin: About project
<p align="justify">This project was written to explore redux as well as react as well as the component style approach. Todos was chosen as an example, you can delete, change and add notes.</p>

---

### :book: Libraries

- react
- react-dom
- react-redux
- redux
- styled-component
- nid

---

### :pizza: Commands

- npm run start: <strong>run in development mode</strong>
- npm run build: <strong>assemble the project</strong>
